# 🚀 Meu Portfólio Pessoal

Este é o meu primeiro projeto de portfólio desenvolvido durante o curso de Desenvolvimento de Sistemas. O objetivo foi criar uma página institucional para apresentar meus serviços como desenvolvedor Front-end, minhas redes sociais e um pouco sobre minha trajetória.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização, layout com Flexbox e uso de variáveis para paleta de cores.
* **Google Fonts:** Fontes *Montserrat* e *Krona One* para uma tipografia moderna.

## 📋 Funcionalidades

- [x] Página inicial com apresentação e chamada para ação.
- [x] Página "Sobre Mim" detalhando minha formação técnica.
- [x] Botões interativos com links para Instagram, GitHub e LinkedIn.
- [x] Layout responsivo e organizado.

## 🎨 Design

O projeto utiliza uma paleta de cores focada no contraste entre o fundo escuro (`#000000`) e o ciano vibrante (`#22D4FD`), garantindo uma leitura clara e um visual tecnológico.
